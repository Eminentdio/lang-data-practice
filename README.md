# LANGUAGE DATA PRACTICE

##### A Delibrate Practice for an Afr. NLP Researcher

*Ismail O. Daud*

This repo is a 3-Month deliberate practice for an African natural prcessing researcher who wants to grow in learning about language data from scratch. It features a beginner-focused curriculum with a mini project after a week.

### Learning Structure
The learning is structured to be taken in four days per week, with a 2-hours focus  per day.

1. #### Day Structure

| Days          | Hours       | Descriptions                    |
|---------------|-------------|---------------------------------|
| Day 1         | 2 Hours     | Guided learning, Reading, tutorial and type out code examples. |
| Day 2         | 2 Hours     | Deconstruction, break the code, change variables,and figure out why. |
| Day 3         | 2 Hours     | Deliberate practice, attempt the weekly project from scratch without looking the tutorial. |
| Day 4         | 2 Hours     | Refinement and Automation. Clean your code, push it to Github, and document your flow.|
